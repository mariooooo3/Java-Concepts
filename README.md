# Concepte Java

Acest repository conține exerciții Java ce acoperă concepte esențiale ale programării orientate pe obiect.

## Cuprins

1. [Încapsularea](1.Încapsularea/lab1Task1)
2. [Constructori, Referințe, Static](2.Constructori, referințe, static)
3. [Agregare și Moștenire](3.Agregare și moștenire/lab3Task1)
4. [Polimorfism](4. Polimorfism/lab4Task1)
5. [Abstractizare](5.Abstractizare)
6. [Structuri de Date](StructuriDeDate)

## Descriere

- **Încapsularea:** Demonstrează cum să protejezi datele și să controlezi accesul.
- **Constructori, Referințe, Static:** Explică utilizarea constructorilor și a cuvântului cheie `static`.
- **Agregare și Moștenire:** Arată cum se realizează moștenirea și agregarea între clase.
- **Polimorfism:** Ilustrează cum obiectele pot avea forme multiple.
- **Abstractizare:** Folosește clase abstracte și interfețe pentru a ascunde implementările.
- **Structuri de Date:** Abordează utilizarea colecțiilor de date în Java.
